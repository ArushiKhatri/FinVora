# FinVora

### Personal Budget & Expense Tracking System

FinVora is a Flask-based personal finance management application designed to help users track income, monitor expenses, manage monthly budgets, and achieve savings goals through a simple and intuitive web interface.

## Overview

Effective financial management begins with understanding where your money comes from and where it goes. FinVora provides a centralized platform for tracking personal finances, helping users make informed budgeting and saving decisions.

## Key Features

| Feature            | Description                                               |
| ------------------ | --------------------------------------------------------- |
| Income Tracking    | Record and manage multiple income sources                 |
| Expense Management | Track daily and monthly expenses                          |
| Budget Planning    | Set monthly spending limits                               |
| Savings Goals      | Define and monitor savings targets                        |
| Financial Summary  | View total income, expenses, and remaining balance        |
| Budget Analysis    | Determine whether spending is within the allocated budget |

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/ArushiKhatri/FinVora.git
```

### Navigate to the Project Directory

```bash
cd FinVora
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```

### Access the Application

```text
http://127.0.0.1:5000
```
